# Проект: Анализ рынка общепита Москвы

### Задача
Провести исследовательский анализ рынка общественного питания Москвы и подготовить рекомендации для открытия нового заведения.

### Основные выводы
- Лидеры рынка по количеству: **кафе** (2376) и **рестораны** (2042)
- Самый насыщенный округ — **ЦАО** (2242 заведения, самый высокий средний чек)
- Самые «сетевые» категории: **булочные** (61 %), **пиццерии** (52 %), **кофейни** (51 %)
- Рейтинг слабо зависит от категории (медиана ≈4.2–4.3), но бары/пабы стабильно выше, фастфуд — ниже
- Высокая цена → высокий рейтинг (логичная зависимость)
- Средний чек падает по мере удаления от центра (в спальных районах на 30–60 % ниже)

### Рекомендации для нового заведения
1. **Бар/паб в САО**  
   - Высокие рейтинги у баров/пабов  
   - Средний чек в САО для категории ≈1573 ₽  
   - Конкуренция ниже, чем в ЦАО

2. **Премиум-булочная в ЦАО**  
   - Мало булочных → низкая конкуренция  
   - Средний чек в ЦАО для булочных ≈1238 ₽ (на 400–600 ₽ выше, чем в спальных районах)
